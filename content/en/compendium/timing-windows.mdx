## Intro
This page is for information about the timing windows of the official SDVX game and its simulators. All of these versions have `CRITICAL`, `NEAR`, and `ERROR` judgements, but only the official game supports `S-CRITICAL`. Note that this information is for chip objects only as the timing windows of laser slams and starts/ends of holds are unknown.

These values are measured in milliseconds (ms) and are **windows** (indicated by ±), meaning a judgement with a timing window of ± 40 ms allows you to hit within 40 ms on both sides (early or late):

## Simulators

### Unnamed SDVX clone (USC) [#usc]
<div className="flex justify-center mt-8">
| Judgement | Window   |
| --------- | -------- |
| CRITICAL  | ± 46 ms  |
| NEAR      | ± 150 ms |
| ERROR     | ± 300 ms |
</div>
These are the default timing windows but they can be adjusted in-game.
