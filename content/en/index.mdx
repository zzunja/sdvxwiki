# Welcome to sdvx.org, the Sound Voltex wiki/guide.

This website was made to try and help people learn more about Sound Voltex
(officially abbreviated as SDVX).

Whether you are new or a veteran, the goal is for this website to contain all
the information about this game in one spot. It can act as a guide for new
players or a refresher on various topics.

On the left you can see all the topics that are on this website. There is also a
search bar if you need to find something specific.

Feel free to share or reference this website anywhere you want.
