<a id="intro"></a>
## 簡介
本質上，模擬器無法提供官方體驗，但它們是免費的，而且可以遊玩 [轉譜](/terminology/termsandvocab#convert) 與玩家自製譜面。

目前有兩款 SDVX 模擬器：Unnamed SDVX clone（USC）與 K-Shoot MANIA（KSM）。兩者都能遊玩相同的譜面，但在功能上有所差異。

<a id="unnamed-sdvx-clone-usc"></a>
## Unnamed SDVX Clone（USC）

<Image src="/setup/simulators/usc_preview.png" alt="USC 預覽" />

[首頁](https://github.com/Drewol/unnamed-sdvx-clone)

<a id="usc-pros-cons"></a>
### 優缺點
**+** [ARS 支援](/compendium/gauge-options#alternative-rate-system-ars)\
**+** 效能優秀\
**+** 歌曲排序與篩選較直觀\
**+** 線上排行榜（僅限轉譜）\
**+** 可在 Windows/macOS/Linux 上遊玩\
**+** 支援直向遊玩\
**+** 練習模式\
**+** 重播系統\
**-** 鏡頭效果可能無法正常運作\
**-** 音效可能不如預期\
**-** 雷射不夠精準

<a id="usc-setup"></a>
### 設定

[下載頁面](https://github.com/Drewol/unnamed-sdvx-clone/releases/)

把檔案解壓縮到電腦上的某個資料夾，然後執行裡面的 `usc-game.exe`。你下載的任何譜面都應該放進第一次開啟遊戲時建立的 `songs` 資料夾（參考 [這裡](#getting-charts)）。

開始遊玩前，建議先調整以下設定：

<Image src="/setup/simulators/usc_settings_1.png" alt="USC 設定 1" />

在 `Input` 分頁中，將 `Button input mode` 和 `Laser input mode` 設成你正在使用的輸入裝置。接著點上方按鈕來修改你的按鍵綁定。
<br/>

<Image src="/setup/simulators/usc_settings_2.png" alt="USC 設定 2" />

在 `Game` 分頁中，將 `Speed mod type` 設為 `MMod`，並調整 `ModSpeed`。對新手來說，建議速度介於 300 到 400 之間。
<br/>

<Image src="/setup/simulators/usc_settings_3.png" alt="USC 設定 3" />

在 `Skin` 分頁中，你可以啟用已安裝的外觀主題，並調整任何該主題特有的設定。

都設定好之後，你就可以開始遊玩了。以下是一些可能用得上的快捷鍵：
  - 在歌曲選擇畫面：
    - `Tab`：搜尋歌曲
    - `F2`：隨機選一首歌
    - `FX-L`：依資料夾篩選，當這個選單開著時按 `START` 可改為依等級篩選
    - `FX-L + FX-R`：開啟遊戲設定選單
    - `BT-B + BT-C`：開啟收藏選單
  - 遊玩時按 `F5` 可快速重開
  - 結算畫面按 `F12` 可截圖

<a id="k-shoot-mania-ksm"></a>
## K-Shoot MANIA（KSM）

<Image src="/setup/simulators/ksm_preview.png" alt="KSM 預覽" />

[首頁](https://www.kshootmania.com/en/)

<a id="ksm-pros-cons"></a>
### 優缺點
**+** 鏡頭與音效運作如預期\
**+** 線上排行榜（僅限自製譜面）\
**-** 效能較差\
**-** 雷射不夠精準（但老實說仍可能比 USC 好）\
**-** 解析度選項有限\
**-** 不支援直向遊玩\
**-** 沒有歌曲搜尋\
**-** 只支援 Windows

<a id="ksm-setup"></a>
### 設定

[下載頁面](https://www.kshootmania.com/en/download.html)

把檔案解壓縮到電腦上的某個資料夾，然後執行裡面的 `kshootmania.exe`。你下載的任何譜面都應該放進 `songs` 資料夾（參考 [這裡](#getting-charts)）。

開始遊玩前，進入 `OPTION` 並調整以下設定：

<Image src="/setup/simulators/ksm_settings_1.png" alt="KSM 設定 1" />

在 `DISPLAY / SOUND` 設定中：
- 將 `DISPLAY LANGUAGE` 設成對應語言
- 將 `BG (WHILE PLAYING)` 設為 `Hide`，以提升遊玩 FPS
- 將 `HIDE "ALL" FOLDER` 設為 `On`，以提升歌曲選擇畫面效能
<br/>

<Image src="/setup/simulators/ksm_settings_2.png" alt="KSM 設定 2" />

在 `INPUT / JUDGEMENT` 設定中：
- 如果你用鍵盤遊玩，將 `INPUT TYPE OF LASER OBJECT` 設為 `Hold key`；如果你用控制器，請參考 [這裡](/guides/ksm-controller-setup)
- 請確認 `ASSIST TICK` 設為 `On`；這個設定有 bug，而且效果是反過來的（on 等於 off，off 等於 on）

回到標題畫面並選擇 `START`。要注意的是，你每次都會被要求登入網路排名系統。
<br/>

<Image src="/setup/simulators/ksm_settings_3.png" alt="KSM 設定 3" />

按住 `BT-D` 會叫出 `Hispeed` 選單，然後按左或右方向鍵切換為固定數值設定。對新手來說，建議速度介於 300 到 400 之間。

都設定好之後，你就可以開始遊玩了。以下是一些可能用得上的快捷鍵：
  - 在歌曲選擇畫面：
    - `FX-L + FX-R`：在字母排序與等級排序之間切換
    - `FX-L` 或 `FX-R`：跳到目前資料夾中的下一個或上一個字母／等級
    - 任意 `BT`：各種遊戲設定
    - 任意 `BT` 搭配 `FX-L` 或 `FX-R`：跳到目前資料夾的開頭或結尾

    - 按住 `START` 可加入或移除收藏
  - 任意 3 個 `BT` 加上 `START` 可快速結束遊戲

安裝主題時，請先確認 KSM 已關閉，然後在 KSM 資料夾中依照以下步驟操作：
  1. 備份你的 `imgs` 和 `se` 資料夾
  2. 將主題提供的 `imgs` 和 `se` 內容解壓縮進對應資料夾，並接受所有覆蓋
  3. 刪除你的 `cache` 資料夾

<a id="which-one-should-i-play"></a>
## 我該玩哪一個？

看完各自的優缺點後，你可能會想：「那為什麼還有人要玩 KSM？」對於選擇 KSM 而不是 USC 的人來說，他們通常認為遊戲引擎本身手感比較好，而這最終才是最重要的。不過，這類玩家通常要不是較有經驗、能感受到差異的人，就是已經玩 KSM 很久、早就習慣它的老玩家。

對新玩家來說，建議先玩 USC，因為它比 KSM 多了很多功能。不過別忘了，這些模擬器都是免費的，所以兩個都試試看也無妨，看看你比較喜歡哪一個。

如果你想玩自製譜面，特別是 [發狂 KSM](/compendium/insane-ksm)，建議玩 KSM。雖然自製譜面也能在 USC 中玩到，但幾乎所有這些譜面在製作時，都是以 KSM 的鏡頭與特效為基礎來設計的。

<a id="getting-charts"></a>
## 取得譜面

轉譜（converts）可以在這裡找到：
  - [SDVX 1-5](https://drive.google.com/drive/folders/1yazwhvpHO-eiEg3VA77BLxIaVy-R1r7G?usp=sharing)，請點右上角的 `Download all`
  - [SDVX EG (6)](https://drive.google.com/drive/folders/1S8gj7-waR4o4o7gU_NZtR3VmsD8UY4ho)，請點右上角的 `Download all`

要注意的是，SDVX EG 與替代版 SDVX 5 合集裡的轉譜，多半都缺少較低的難度，因為轉譜是一個需要人工操作、相當耗時的過程，而且是由志願者完成的。

自製譜面可以在 [ksm.dev](https://ksm.dev/) 找到。幾件事要注意：
  - 這個網站只收 **自製譜面**，不接受也找不到轉譜上傳檔
  - 譜面品質無法保證，使用者可以自由上傳自己做的任何譜面
  - 雖然也能找到完整難度分布，但大多數自製譜面等級都偏高，因為它們的作者通常更擅長這款遊戲，或更投入其中
  - 譜面等級是依感覺而定，不是固定公式，所以有些自製譜面可能會比標示等級更簡單或更難

要安裝譜面，請將它們解壓縮並放進 USC/KSM 安裝目錄的 `songs` 資料夾。建議不要更動資料夾的架構，因為遊戲內會用資料夾來做分類。例如你要安裝 `Butterfly Twist` 的譜面，路徑應該長這樣：
  - `/kshootmania/songs/SOUND VOLTEX V VIVID WAVE/butterfly_twist`

而不是這樣：
  - `/kshootmania/songs/butterfly_twist`

KSM 也支援子資料夾來進一步整理歌曲，但 USC 不支援。
