## 找到 SDVX 機台

就價格與便利性而言，最便宜也最方便的玩法，通常就是去街機。

<a id="valkyrie-vs-nemsys-model"></a>
### Valkyrie 與 Nemsys 機型

目前執行 SDVX 的機台有兩種：Valkyrie Model（女武神筐/新筐）與 Nemsys Model（舊筐）。兩者差異可參考 [Valkyrie 與 Nemsys 機型](/specifications/valkandnemsys)。簡單來說，Valkyrie Model 是較新的機台，而 Nemsys Model 則是較舊的機台。

- Valkyrie 機台配備 43 吋、120Hz 螢幕與觸控螢幕
- Nemsys 機台配備 32 吋、60Hz 螢幕

<a id="official-konami-servers"></a>
#### 官方 Konami 伺服器

如果你想玩官方 Konami 伺服器，就要找 Valkyrie Model。雖然也有一些 Valkyrie Model 並不接官方伺服器，但數量非常少。要注意的是，在菲律賓與南韓，仍有使用官方伺服器的 Nemsys Model 機台。你可以用 [zenius-i-vanisher](https://zenius-i-vanisher.com/v5.2/arcades.php) 找附近的街機。搜尋 `SOUND VOLTEX EXCEED GEAR (Valkyrie Model)` 來找有 Valkyrie Model 機台的店家。

<Image src="/arcade/zenius.png"/>

[VMSearch](https://www.vmsearch.net/) 也會列出所有存在的 Valkyrie Model 機台。

在臺灣可以用 [Music Game Map](https://mgm.wind-chime.info/)。


<a id="non-official--private-servers"></a>
#### 非官方／私人伺服器

如前所述，除了菲律賓與南韓以外，任何使用 Nemsys Model 機台的街機都不會跑官方伺服器（例如 Eagle）。有些店家會把 Nemsys Model 升級成 32 吋、120Hz 螢幕，並更換按鈕與旋鈕，讓手感更接近 Valkyrie Model，這點相當不錯。你也可以用 [zenius-i-vanisher](https://zenius-i-vanisher.com/v5.2/arcades.php) 找附近的街機。搜尋 `SOUND VOLTEX EXCEED GEAR` 就能找到使用 Nemsys Model 的店家。

<a id="playing-in-an-arcade-for-the-first-time"></a>
## 初次遊玩街機 SDVX

<a id="get-an-e-amusement-card"></a>
### 取得 e-amusement 卡

e-amusement 卡用來儲存你的遊玩紀錄，也能在 SDVX 以外的其他遊戲中使用。街機通常可以在櫃台購買 e-amusement 卡。

<Image src="/arcade/eamusementcard.png"/>

盡量挑有右下角 Amusement IC 標誌的卡片。現在大多數卡片都有這個標誌，舊款卡片則已經幾乎淘汰。如果你買不到 e-amusement 卡，只要有這個標誌的卡片都能用，例如 Aime、Banapass 和 NESiCA。

<a id="creating-an-account"></a>
## 建立帳號

<Callout type="warning" emoji="⚠️">
  如果你有玩過 Konasute，**強烈**建議把 Konasute 資料轉移到那張卡上，而不是在卡片上建立新的資料。如果卡片已經有資料，你就無法把 Konasute 資料和街機資料合併。請參照 [家用版 - Konasute](/setup/konasute#linking-arcade-and-konasute-data) 章節。
</Callout>

拿到 e-amusement 卡之後，前往 SDVX 機台並將卡片輕觸感應區。接著系統會提示你建立資料。

<Image src="/arcade/e-amuse-scan.png"/>

<a id="creating-a-profile-and-playing-your-first-credit"></a>
## 建立資料並開始遊玩

<Callout>
  這裡很多資訊都來自 [這份指南](https://voltexes.com/arcade-quick-start-guide-eg/)。若你感到困惑，可以去看看原始指南。
</Callout>
刷卡後，你需要建立自己的資料。

<Image src="/arcade/start1.jpg"/>

系統會詢問你要轉移資料還是建立新資料。既然你沒有可轉移的資料，直接選擇建立新資料即可。

<Image src="/arcade/start2.jpg"/>

接著你要設定一組 PIN。這組 PIN 之後也會用在你玩的其他所有 BEMANI 遊戲上。

<Image src="/arcade/start3.jpg"/>

然後你要建立名字。使用旋鈕選字，按 `START` 選擇字元。名字選好後，選擇 END 按鈕並完成資料建立。

建立資料後，系統會出現很多選項。初次遊玩建議選 `NORMAL`，再選 `STANDARD START`。如果沒有 `STANDARD START`，就選 `LIGHT START`。`STANDARD START` 較好，因為它能使用 S-Critical、[ARS](/compendium/gauge-options/#alternative-rate-system-ars)，也會移除等級限制等功能。這些模式的詳細說明可以看 [這裡](/compendium/game-modes#normal)。

選定模式後，你會進入 `MY ROOM` 畫面。在這裡，你可以更換 navigator（成績畫面上會出現的角色）、appeal card（大致上就是頭像）、查看 tama adventure（歌曲解鎖進度系統），如果 arena shop 有開也能看到。最後，選擇 `START` 就能開始你的這一局。

<a id="linking-your-e-amusement-to-your-konami-account"></a>
## 將 e-Amusement 連結到你的 Konami 帳號
<Callout>
  如果你不是在官方伺服器上玩，這個設定不適用。雖然流程大致相同，但你多半得改走該伺服器自己的網站。
</Callout>

如果你是在官方伺服器上遊玩，建議把 e-Amusement 卡連結到你的 Konami 帳號。先建立或登入你的 Konami 帳號，再前往 [這個網站](https://p.eagate.573.jp/gate/eapass/attach.html)。（關於建立帳號的更多資訊可參考 [這裡](/setup/konasute#creating-a-konami-account)）

在 `カードナンバー` 欄位中輸入 e-amusement 卡背面的數字。接著在下方標示為 `暗証番号` 的欄位輸入你剛剛設定的 4 位數 PIN。最後再往下找到 `登録する` 按鈕，把 e-amusement 卡綁定上去。

如果你還是有點混亂，可以參考這份 [e-Amusement 指南](https://3icecream.com/tutorial/check-eamuse)。雖然它是以 DDR 為例，但 e-Amusement 的設定方式同樣適用於所有 BEMANI 遊戲。
