## Blaster Gate [#blaster-gate]
Blaster Gate 歌曲收錄在 Blaster Gate 資料夾中。被 Blaster Gate 鎖定的歌曲，在主選曲畫面會以鎖頭標示。若要開啟資料夾並解鎖其中的歌曲，必須填滿 Blaster 量表；歌曲結算畫面會顯示目前累積進度，如下圖。

<div className="flex items-center justify-center mt-3">
 <img src='/compendium/unlock-systems/blastergauge.webp' width="400"/>
</div>

量表達到 100% 後，就能進入 Blaster Gate 資料夾遊玩想解鎖的歌曲。除非持有 Blaster Pass，否則必須通關譜面才能解鎖。通關較高難度也會一併解鎖較低難度。

無論通關或失敗，Blaster 量表都會重設為 0。

如果累積 20 次都未能通關該歌曲，歌曲會自動解鎖，無論通關或失敗。

## Variant Gate [#variant-gate]
<Image src="/compendium/unlock-systems/variant_charts.webp"/>

Variant Gate 是《SOUND VOLTEX》Exceed Gear 新增的解鎖系統。

要解鎖前 3 首歌曲，必須依目標譜面的 [譜面雷達圖](/compendium/effect-radar) 填充 Variant 量表（`VARIANT POWER`）。在第一個 Variant Gate 中，前 3 首歌曲分別對應以下 3 組譜面雷達圖屬性：
- TRICKY 與 NOTES
- HAND-TRIP 與 TSUMAMI
- ONE-HAND 與 PEAK

通關譜面後，Variant 量表會依該譜面的譜面雷達圖填充。例如遊玩 `NOTES` 數值最高的譜面，就會填充 TRICKY 與 NOTES 屬性。

每組雷達圖屬性都有對應歌曲。例如在第一個 Variant Gate 資料夾中，填充 `ONE-HAND` 或 `PEAK` 屬性就能解鎖 **Who then no 灯**。

可以使用 [譜面雷達圖資料夾](/startingout/gameplay-screens#song-select-folders) 依雷達類型篩選譜面。

填滿量表後，只有在 **EX-TRACK** 中才能進入 Variant 資料夾並選擇譜面。Variant 資料夾只會在 EX-TRACK 中顯示。不同的 RANK 代表不同的量表難度（見 [Variant Rate 詳細表格](/compendium/gauge-options#variative-rate-breakdown)）。初次選擇譜面時，會預設使用 `RANK V` 的血條。這是最困難的血條，無法回復血量，且打出 Critical 判定時會扣血。

若挑戰失敗，下次遊玩該譜面時會降低一個 RANK。例如在 `RANK V` 失敗後，再次挑戰時會改為 `RANK IV`，最低會降至 `RANK I`。

無論使用哪個 RANK，通關後都會解鎖該譜面及所有較低難度。

要解鎖 20 級歌曲，必須通關前 3 張譜面的 MXM 獲得「刻印」。通關譜面後，可以選擇要挑戰的難度；血條一開始仍預設為 `RANK V`。

前兩個 Variant Gate（神凪 與 Everlasting Message）的刻印會保留。其他 Variant Gate 在失敗後會失去刻印，必須再次以任意 RANK 通關所有前置譜面才能取得。之後可以從 `RANK IV` 開始挑戰；若在 `RANK IV` 失敗，下次就能從 `RANK III` 挑戰，依此類推。

`VARIANT POWER` 最多可超量填充至 500%。累積到至少 200% 後，就可以開始選擇較低的初始血條 RANK。注意挑戰時就會消耗所有 `VARIANT POWER`。

## 活動解鎖 [#event-unlocks]
活動限定歌曲有特定的開放期間，之後會不定期復刻。有多種解鎖方式。

以下僅介紹最常見的活動，並不包含所有活動；偶爾也會推出全新的歌曲解鎖活動。

### 集章 [#stamp]
這是最常見的活動。每首歌曲都有所需的集章數，並在一道結束時獲得集章。Standard Start 可獲得 1 枚，其他模式可獲得 2 枚。（沒有固定規律，但通常費用較高的模式會給予較多。）

### Arena 商店 [#arena-shop]
Arena 商店會不定期開放，販售不同歌曲，有時也會販售領航員。商品大多是 XCD。購買歌曲需要 AP；AP 只會在 Arena 商店開放期間取得，並依選擇的模式在一道結束時獲得。

### Triple Tribe 等跨遊戲聯動 [#triple-tribecrossover]
Triple Tribe 等跨遊戲聯動活動通常是 SDVX、DDR 與 IIDX 之間的合作，不一定每次都包含所有遊戲。活動會為每款遊戲提供一條量表，填滿後即可解鎖歌曲。必須遊玩不同遊戲才能解鎖特定歌曲，且所有量表都填滿才能解鎖最後一首歌曲。

### Omega Dimension [#omega-dimension]
Omega Dimension 的運作方式與 Blaster Gate 相同，只是位於不同的資料夾。
### HEXA DIVER [#hexadiver]
HEXA DIVER 是 Exceed Gear 的解鎖系統之一。Blaster 量表達到 100% 後即可進入此資料夾。每個 HEXA DIVER 都是獨立資料夾，包含 7 張譜面，最後一張為 20 級譜面。

首次進入 HEXA DIVER 時，只有 3 首歌曲可供解鎖；解鎖後才會出現其他歌曲。

每首歌曲有 10000 HP。達到 1000（NOV）、2500（ADV）、5000（EXH）與 10000（MXM）傷害門檻後，便會解鎖對應難度。遊玩較高難度可以造成更多傷害，但最多只能解鎖目前遊玩的難度。例如遊玩 EXH 時，無法造成更多傷害，也無法超過 5000 HP 的 EXH 門檻，因此不能解鎖 MXM。

選擇譜面後，還能選擇血條難度，以 HEXA RANK（簡稱 HR）表示。難度越高，造成的傷害越多，但也越難通關。一般來說，HR4 約等同於 EXCESSIVE，HR5 則更加困難。

Akasha 之前的 HEXA DIVER 有部分專屬改動，具體包括 `幻想御伽篇` 至 `彩虹の銀幕篇` 的 HEXA DIVER 1～9：
- **HEXA BLASTER**：同一道內可以多次遊玩 HEXA DIVER。
- **HEXA CONTINUE**：歌曲失敗時不會消耗 Blaster Energy。
- **HEXA OVER DRIVE**：每通關一次 HEXA DIVER 歌曲，傷害倍率就會增加；每天重設。

### Tama Adventure [#tama-adventure]
Tama Adventure 是 Exceed Gear 的解鎖系統。開始一道前，可以選擇想解鎖的歌曲；進入選曲前，也能在 MY ROOM 中變更目標歌曲。每首歌曲都有解鎖所需的點數，可以透過完成任務或遊玩取得。達到不同點數門檻後，會依序解鎖較低難度。

一道結束時，系統會統計點數並用於解鎖歌曲。不同難度所需的點數不同；Blaster Pass、Blaster Start 與 Premium Time 都能獲得更多點數。

每首歌曲都有可取得額外點數的任務。選曲時，畫面底部會顯示任務內容；由於內容是日文，建議參考 [解鎖指南](https://docs.google.com/spreadsheets/d/1VBYWSUaKqHUOyfCtvkHFYBSnEPwxhTh8-m4UlU8zUL0/edit?gid=594247546#gid=594247546)。

任務不是完成後直接給予固定點數，而是提高單一首歌曲可獲得的點數。

每首歌曲基本可獲得 3 點，另加已完成的任務數量。例如完成 3 個任務後遊玩一首歌曲，該歌曲可獲得 `6` 點。
